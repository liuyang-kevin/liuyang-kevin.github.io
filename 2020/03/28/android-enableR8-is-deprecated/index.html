android.enableR8' is deprecated

更新项目时候发现。

build.gradle: The option ‘android.enableR8’ is deprecated and should not be used anymore.

Android代码压缩工具R8

R8 这一项工具的目的是让代码缩减过程更快和更有效率。

  1. dex生成过程
  2. 2015年，谷歌 Android 团队引入了编译器 Jack 和 Jill
  3. 2017年，使用新的 D8 编译器，
  4. R8 是 D8 的衍生产品，旨在集成 ProGuard 和 D8 的功能：

R8 一步到位地完成了所有的缩减（shrinking），去糖（desugaring）和 转换成 Dalvik 字节码（dexing ）过程。

R8 和当前的代码缩减解决方案 Proguard 相比，R8 可以更快地缩减代码，同时改善输出大小。
